## 🚀 Sobre o Projeto

Este portfólio foi construído com foco total em **performance e boas práticas**, aplicando os seguintes conceitos:

- **Estrutura Semântica:** Uso correto de tags HTML5 para melhor SEO e acessibilidade.
- **CSS Modular:** Estilos organizados por arquivos específicos, facilitando a manutenção e organização (BEM-like).
- **Vanilla JavaScript:** Manipulação de DOM e lógica de interface sem dependência de frameworks externos.
- **Responsividade:** Design adaptável para qualquer tamanho de tela (Mobile First).
- **Experiência do Usuário (UX):** Transições suaves e interfaces intuitivas para uma navegação fluida.

### ⚡ O que o site apresenta:

- **Animação de Digitação:** Efeito _typewriter_ dinâmico no título principal da Home.
- **Slider Interativo:** Navegação personalizada e automática na seção "Sobre Mim".
- **Dark/Light Mode:** Alternância de tema (claro e escuro) com manipulação de classes.
- **Cards Dinâmicos:** Projetos renderizados via JS a partir de uma base de dados interna (Array de objetos).
- **Modais Informativos:** Detalhes dos projetos exibidos através da API nativa `<dialog>`.
- **Formulário de Contato:** Integrado com _FormSubmit_ para envio real de mensagens.

---

## 🎯 Diferenciais Técnicos

Diferente de portfólios estáticos, este projeto foca em **programação funcional e dinâmica**:

- **Manipulação de DOM:** Criação dinâmica de cards de projetos via JavaScript, permitindo fácil manutenção sem alterar o HTML.
- **Modo Escuro (Dark Mode):** Implementação de persistência de estilo e manipulação de classes CSS.
- **Lógica de Componentização:** Uso de `modais` nativos (elemento `<dialog>`) para exibição detalhada de projetos.
- **UX/UI:** Efeito _Typewriter_ no cabeçalho e Sliders customizados para apresentação pessoal.

## 📂 Organização do Código

O projeto segue o padrão de **CSS Modular**, onde cada seção possui seu próprio arquivo de estilo, facilitando a escalabilidade:

```text
├── 📁 css/             # Estilos separados por componentes/seções
├── 📁 image/           # Assets e thumbnails dos projetos
├── 📁 curriculo/       # Arquivo PDF para download
├── 📄 index.html       # Estrutura principal
├── 📄 script.js        # Toda a lógica do Portfólio
└── 📄 projects.html    # Galeria expandida de projetos
```
